# Dhruv Technology Solutions LLC - Website

Professional B2B website for AI and IT consulting services.

## Overview

This website showcases Dhruv Technology Solutions LLC's comprehensive AI consulting and software development services. The design emphasizes expertise, security, and forward-thinking solutions for enterprise clients.

## Website Structure

### Pages

1. **Homepage (`index.html`)**
   - Compelling value proposition headline
   - Service overview cards (4 core services)
   - "Why Choose Us" section highlighting key differentiators
   - Clear CTAs for booking free AI strategy sessions

2. **About Us (`about.html`)**
   - Expert positioning as specialized AI consultant
   - Technical expertise and business acumen
   - Philosophy and approach to AI transformation
   - 4-phase implementation methodology
   - Commitment to personalized service

3. **Services (`services.html`)**
   - Detailed descriptions of all 8 services:
     - AI Consulting & Strategy
     - Predictive Analytics & Data Services
     - AI-Driven Cybersecurity
     - Intelligent Automation & RPA
     - Generative AI Development
     - Conversational AI & Customer Support
     - Computer Vision Services
     - AI-Powered HR & Recruitment
   - Focus on ROI and business outcomes
   - Specific deliverables and business impact for each service

4. **Contact (`contact.html`)**
   - Professional contact form
   - Free AI Strategy Consultation CTA
   - FAQ section
   - Email contact information

## Design Features

- **Modern, Professional Aesthetic**: Clean design with Inter font family
- **Conversion-Focused**: Multiple strategic CTAs throughout
- **Mobile Responsive**: Adapts to all screen sizes
- **Brand Colors**: 
  - Primary: Blue (#2563eb)
  - Secondary: Dark slate (#0f172a)
  - Accent colors for visual hierarchy

## Key Messaging

- **Tone**: Professional, confident, tech-forward
- **Positioning**: Expert consultant with personalized attention
- **Value Proposition**: Enterprise-grade AI solutions with measurable ROI
- **Differentiation**: Direct senior-level expertise, no junior teams

## Technical Stack

- HTML5
- CSS3 (Custom styling, no frameworks)
- Vanilla JavaScript (minimal, for form handling)
- Google Fonts (Inter)

## Setup Instructions

1. Open `index.html` in a web browser to view the website
2. All pages are linked via navigation menu
3. No build process required - static HTML/CSS

## Form Integration

The contact form currently shows an alert message. To make it functional:

1. Set up a backend service (e.g., Node.js/Express, PHP)
2. Configure email service (SendGrid, AWS SES, etc.)
3. Update the form submission handler in `contact.html`
4. Add proper validation and security measures

## Customization

To customize for production:

1. Replace `info@dhruvtechsolutions.com` with actual email
2. Add real company contact information
3. Integrate analytics (Google Analytics, etc.)
4. Set up form backend
5. Add SSL certificate for HTTPS
6. Consider adding a blog section for thought leadership
